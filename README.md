
This repository contains multiple projects in Natural Language Processing (NLP), Speech Recognition, Deep Learning, and Text Generation.

---

## TASK 1: NLP Text Summarization
Automatic text condensing using LSA, Luhn, and TextRank algorithms.  
Takes long documents and produces concise summaries.

---

## TASK 2: Speech to Text Recognition
Real-time speech recognition with Tkinter GUI.  
Converts audio from microphone to text using Google Cloud API.

---

## TASK 3: Neural Style Transfer
Deep learning project using VGG-19 architecture.  
Applies artistic style from one image to content of another.  
Uses content and style loss optimization.

---

## TASK 4: Text Generation
Advanced language model for generating coherent paragraphs.  
Customizable temperature, top-k, and top-p sampling.  
Creates contextually relevant text from prompts.
